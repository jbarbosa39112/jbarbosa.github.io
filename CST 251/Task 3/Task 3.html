Introduction

HTML (Hyper Text Markup Language), a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.

Basic Tags

An HTML tag is commonly defined as a set of characters constituting a formatted command for a Web page. At the core of HTML, tags provide the directions or recipes for the visual content that one sees on the Web.

The following are some of the most common basic tags:

  • The <a> tag defines a hyperlink, which is used to link from one page to another.

  • The <br> tag creates a line break.

  • The <h1> tag is used for titles, there are also 5 other h tags that vary sizes.

  • The <p> tag defines a paragraph

  • The <Strong> tag is used to make text bold.

Form Tags

The form tag is used to create an HTML form for user input.

The following are some of form tags used:

  • The <input type="text" name="..."> tag can be used to request a username.

  • The <input type="text" number="..."> tag can be used to let the user choose a number.

  • The <radio> tag can be used so that the user can choose between subjects like male or female.

  • The <button> send </button> tag is used to create a clickable button.
